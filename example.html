Input File with good visual

 

Clean - Enable - Disable

 

Clean - Enable - Disable

 

Using selector jquery for all files

Clean All - Enable All - Disable All

 

Create

// with default options
$('input:file').file();

// with options
$('input:file').file({
       width: 170,
       height: 26,
       label: 'File',
       tip_remove: 'Remove File'
});

Clean All

$('input:file').file('clean');

Enable All

$('input:file').file('enable');

Disable All

$('input:file').file('disable');
